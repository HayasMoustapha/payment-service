# Payment Service

Service de paiements pour Event Planner - Gestion des transactions, facturation et abonnements.

## Fonctionnalités

- Paiements événements
- Achats de tickets
- Facturation
- Remboursements
- Abonnements
- Intégration Stripe/PayPal

## Stack Technique

- Node.js + Express
- Stripe API
- PostgreSQL
- Redis (cache)

## Installation

```bash
npm install
npm start
```

## Développement

```bash
npm run dev
```

## Docker

```bash
docker-compose up -d
```
